# Каталог помощников локализации

Этот каталог содержит классы-помощники локализации, используемые библиотекой Eleanor PHP Library. Помощники локализации содержат языковую логику, которую нельзя удобно выразить простыми строками перевода. Например: форматирование дат, множественные формы слов, транслитерация и другие операции, зависящие от грамматики конкретного языка.

## Пространство имен

Все классы-помощники в этом каталоге находятся в пространстве имен `Eleanor\Classes\L10n`. Например:

```php
namespace Eleanor\Classes\L10n;

class Ru
{
    // ...
}
```

Основной фасад `Eleanor\Classes\L10n` выбирает класс-помощник по текущему коду языка и вызывает его статические методы.

## Именование файлов

Каждый файл соответствует одному языковому помощнику и должен называться по коду языка в нижнем регистре:

```text
en.php
ru.php
```

Имя класса должно использовать тот же код языка в PascalCase:

```php
Eleanor\Classes\L10n\En
Eleanor\Classes\L10n\Ru
```

## Обязательный интерфейс

Языковые помощники должны реализовывать `Eleanor\Interfaces\L10n`. Сейчас интерфейс требует метод `Date()`:

```php
static function Date(int|string $d, \Eleanor\Enums\DateFormat $f): string;
```

Метод должен форматировать timestamp или строку даты/времени согласно переданному варианту `DateFormat`.

## Дополнительные методы

Помощники могут объявлять дополнительные методы, специфичные для языка, например:

- `Plural()` для множественных форм слов;
- `Translit()` для языковой транслитерации;
- внутренние методы для форматирования дат и названий месяцев.

Такие методы не входят в общий интерфейс, пока они явно не нужны универсальному коду библиотеки.

## Добавление нового языка

Чтобы добавить новый помощник:

1. Создайте файл с именем кода языка, например `de.php`.
2. Объявите класс в пространстве имен `Eleanor\Classes\L10n`, например `De`.
3. Реализуйте `Eleanor\Interfaces\L10n`.
4. Добавьте обязательный метод `Date()` и нужные языковые методы.

Пример:

```php
<?php
namespace Eleanor\Classes\L10n;

use Eleanor\Enums\DateFormat;

class De extends \Eleanor\Basic implements \Eleanor\Interfaces\L10n
{
    static function Date(int|string $d=0, DateFormat $f=DateFormat::HumanDateTime): string
    {
        // Форматирование для конкретного языка.
    }
}
```
